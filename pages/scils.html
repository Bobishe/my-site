Навыки:

  • PHP

    язык программирования, который используется для создания динамических веб-страниц и веб-приложений

  • JS

    Позволяет создавать интерактивные элементы на веб-страницах, такие как кнопки, формы, анимации, слайдеры, меню и многое другое

  • Boostrap

    фреймворк для фронтенд-разработки, который предоставляет набор инструментов и компонентов для создания современных и адаптивных веб-интерфейсов

  • Laravel

    бесплатный фреймворк для веб-разработки, написанный на языке PHP

  • jQuery

    библиотека JavaScript, которая упрощает работу с HTML-документами, обеспечивает удобный доступ к элементам DOM, управляет событиями и анимацией, и позволяет создавать интерактивные веб-страницы

  • Node.js

    платформа для выполнения JavaScript на стороне сервера, которая позволяет создавать масштабируемые и быстрые сетевые приложения

  • SQL

    язык программирования, который используется для работы с реляционными базами данных
